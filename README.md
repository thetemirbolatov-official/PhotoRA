# PhotoRA - Кинематографичный фоторедактор

Профессиональный фоторедактор с кинематографичными фильтрами. Превращайте обычные фотографии в шедевры одним кликом.

---

## О программе

PhotoRA - это мощный инструмент для обработки фотографий, созданный для тех, кто ценит качество и стиль. Программа предлагает коллекцию профессиональных кинематографичных фильтров, которые придают вашим снимкам особую атмосферу и глубину.

## Как использовать

1. Запустите программу PhotoRA
2. Выберите фотографию (кнопка "ВЫБРАТЬ ФОТО" или перетащите файл в окно программы)
3. Нажмите на любой фильтр для применения эффекта
4. Сохраните результат (кнопка "СОХРАНИТЬ")

Фотографии сохраняются в оригинальном разрешении в папку Documents/PhotoRA

## Системные требования

- Операционная система: Windows 7, 8, 10, 11 (32/64 бит)
- Оперативная память: от 2 ГБ
- Свободное место на диске: от 100 МБ

## Установка

1. Скачайте установочный файл PhotoRA_Setup.exe
2. Запустите установку
3. Следуйте инструкциям установщика
4. Запустите программу через ярлык на рабочем столе или в меню Пуск
